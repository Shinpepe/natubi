# 📊 나투비 (나의 투자 비서)

실적과 데이터로 읽는 투자 시장.

나투비는 투자 초보자를 위한 데일리 투자 브리핑 서비스입니다.
매 거래일 장 마감 후 시장 데이터를 자동으로 수집하고 분석해서,
"오늘 시장이 어떤 분위기인지, 어떤 종목을 눈여겨볼 만한지"를 한 화면에 정리해 줍니다.

> 접속 주소: https://shinpepe.github.io/natubi/

## 무엇을 해주나요

**🌐 시장 현황**
KOSPI 흐름을 종합한 공포-탐욕 지수로 시장의 온도를 보여줍니다.
섹터별 자금 흐름(트리맵), 환율과 글로벌 지표 추이도 함께 제공합니다.

**🔍 종목 추천**
좋은 기업(DART 실적 검증) → 오르는 흐름 → 매수 타이밍, 세 관문을 모두 통과한 종목만 추천합니다.
추천마다 이유를 한 줄로 설명하고, 관련 뉴스 헤드라인을 붙여줍니다.
조건이 엄격해서 추천이 없는 날도 있습니다. 그것도 하나의 신호입니다.

**📋 포트폴리오**
보유 종목을 등록하면 종가 기준 수익률과 함께 추세 신호(데드크로스, 추천 손절가)를 계산해
매도 검토가 필요한 종목을 알려줍니다.

**💰 모의 투자**
가상 자본 1,000만 원으로 매매를 연습할 수 있습니다.
평단가와 실현손익이 자동 관리되고, 자산 추이 그래프로 성과를 돌아볼 수 있습니다.

**🧮 투자 계산기 · 💣 미니 게임**
물타기, 복리, 배당 계산기와 쉬어가는 지뢰찾기.

## 어떻게 동작하나요

- 매 거래일 장 마감 후 데이터가 자동으로 갱신됩니다.
- 별도의 서버 없이 정적 웹페이지로 동작하며, 회원가입이나 로그인이 없습니다.
- 포트폴리오와 모의투자 기록은 사용자 본인의 브라우저에만 저장됩니다. 외부로 전송되지 않습니다.

## 유의사항

나투비가 제공하는 모든 지표와 추천은 참고용입니다.
일 단위 확정 데이터를 기반으로 하며 실시간 시세가 아닙니다.
최종 투자 판단과 그에 따른 책임은 투자자 본인에게 있습니다.
